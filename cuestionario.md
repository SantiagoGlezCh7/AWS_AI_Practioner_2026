## Cuestionario 

¿Qué opción describe MEJOR el aprendizaje supervisado en el aprendizaje automático?

A. Aprender a partir de pares de entrada-salida etiquetados

B. Aprender a partir de datos no etiquetados

C. Aprender mediante ensayo y error

D. Aprender agrupando puntos de datos similares


<details>
<summary>Ver respuesta</summary>

El aprendizaje supervisado es un tipo de aprendizaje automático en el que el algoritmo se entrena con un conjunto de datos etiquetados. 

Esto significa que, para cada entrada de los datos de entrenamiento, existe una salida correcta o etiqueta correspondiente. El algoritmo aprende a asignar entradas a salidas analizando estos pares de entrada-salida.

Durante el entrenamiento, el modelo realiza predicciones sobre los datos de entrada y las compara con las etiquetas reales, ajustando sus parámetros para minimizar la diferencia entre sus predicciones y las etiquetas verdaderas.

Este proceso permite al modelo aprender patrones y relaciones en los datos, que luego puede utilizar para realizar predicciones sobre datos nuevos y desconocidos.

Las demás respuestas son incorrectas por las siguientes razones:
* **B.** El aprendizaje a partir de datos sin etiquetar describe el aprendizaje no supervisado, no el aprendizaje supervisado.
* **C.** El aprendizaje mediante ensayo y error es más característico del aprendizaje por refuerzo, no del aprendizaje supervisado.
* **D.** El aprendizaje mediante la agrupación de puntos de datos similares es una técnica utilizada en el aprendizaje no supervisado, no en el aprendizaje supervisado.

¿Qué servicio de IA gestionado de AWS está diseñado para convertir el habla en texto?

A. Amazon Lex

B. Amazon Polly

C. Amazon Transcribe

D. Amazon Textract


<details>
<summary>Ver respuesta</summary>

La respuesta correcta es la C: Amazon Transcribe.

Amazon Transcribe es un servicio de inteligencia artificial gestionado por AWS diseñado específicamente para convertir el habla en texto. Utiliza modelos avanzados de aprendizaje automático para reconocer automáticamente el habla en archivos de audio o transmisiones en directo y convertirla en transcripciones de texto precisas. Amazon Transcribe admite varios idiomas y puede identificar a diferentes interlocutores en una conversación. Este servicio resulta especialmente útil para aplicaciones como la transcripción de llamadas de atención al cliente, la generación de subtítulos para vídeos o la creación de archivos de contenido de audio en los que se puedan realizar búsquedas.

Las demás respuestas son incorrectas por las siguientes razones:

A. Amazon Lex se utiliza para crear interfaces conversacionales mediante voz y texto, no para la conversión de voz a texto.

B. Amazon Polly convierte texto en voz, no voz en texto.

D. Amazon Textract extrae texto y datos de documentos escaneados, no de audio.

¿Cuál es la opción que MEJOR describe el papel del aprendizaje profundo en el contexto de la inteligencia artificial y el aprendizaje automático (IA/ML)?


A. El aprendizaje profundo es una técnica tradicional de aprendizaje automático que no utiliza redes neuronales.

B. El aprendizaje profundo es sinónimo de IA y abarca todas las técnicas de aprendizaje automático.

C. El aprendizaje profundo es un campo independiente, sin relación con la IA ni con el aprendizaje automático.

D. El aprendizaje profundo es un subconjunto del aprendizaje automático que utiliza redes neuronales con múltiples capas.

<details>
<summary>Ver respuesta</summary>

La respuesta correcta es la D. El aprendizaje profundo es un subconjunto del aprendizaje automático que utiliza redes neuronales con múltiples capas.

El aprendizaje profundo representa un subconjunto especializado del aprendizaje automático que utiliza redes neuronales artificiales con múltiples capas (también denominadas capas ocultas). Estas redes neuronales procesan los datos a través de capas sucesivas, en las que cada capa aprende características cada vez más complejas a partir de los datos de entrada. Esta arquitectura permite a los modelos de aprendizaje profundo aprender automáticamente representaciones de los datos con múltiples niveles de abstracción, lo que los hace especialmente eficaces para tareas complejas como el reconocimiento de imágenes, el procesamiento del lenguaje natural y el reconocimiento de voz.

Las demás respuestas son incorrectas por las siguientes razones:

A. El aprendizaje profundo utiliza específicamente redes neuronales con múltiples capas.

B. El aprendizaje profundo no es sinónimo de IA. Se trata de un enfoque especializado dentro del campo más amplio del aprendizaje automático.

C. El aprendizaje profundo está directamente relacionado con la IA y el aprendizaje automático como un subconjunto de estos campos, no como un campo independiente y sin relación con ellos.
